🆘 TROUBLESHOOTING

GUI won't launch?
  → Make sure you're in the correct folder
  → Ensure AddSub-Multi-6.0.bat is in the same folder
  → Try running: pip install customtkinter tkinterdnd2

Processing not working?
  → The GUI uses the batch file underneath
  → Check batch file dependencies:
    • ffmpeg (in PATH)
    • ffprobe (in PATH)
    • python (in PATH)
    • subliminal (pip install subliminal)

Multiple instances not queuing?
  → Check %LOCALAPPDATA%\AddSub-Multi\ exists
  → Try running as administrator
