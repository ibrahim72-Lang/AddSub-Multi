  NEW FEATURE ADDED
───────────────────────────────────────────────────────────────────────

✅ ADDITIVE FILE DROPPING
   → When you drop or browse for files, they are ADDED to the list
   → Previous behavior: New files REPLACED the entire list
   → New behavior: New files are APPENDED to existing files
   → Build up your batch by adding files one at a time or in groups!

───────────────────────────────────────────────────────────────────────
  HOW IT WORKS
───────────────────────────────────────────────────────────────────────

SCENARIO 1: Adding Files One by One
   1. Drop "Movie1.mp4" → List shows: [Movie1.mp4]
   2. Drop "Movie2.mp4" → List shows: [Movie1.mp4, Movie2.mp4]
   3. Drop "Movie3.mp4" → List shows: [Movie1.mp4, Movie2.mp4, Movie3.mp4]
   ✓ Files accumulate in the list!

SCENARIO 2: Adding Multiple Files at Once
   1. Drop 3 files → List shows: [Movie1.mp4, Movie2.mp4, Movie3.mp4]
   2. Drop 2 more → List shows: [Movie1.mp4, ... Movie5.mp4]
   ✓ All files kept, new ones added!

SCENARIO 3: Using Browse Button
   1. Browse for files → Select 2 files → Added to list
   2. Browse again → Select 3 more → Added to existing list
   ✓ Total: 5 files in the list

SCENARIO 4: Mixing Files and Folders
   1. Add files → List shows files
   2. Add folder → List CLEARS and shows folder
   ✓ Folders always replace files (by design)

───────────────────────────────────────────────────────────────────────
  SMART FEATURES
───────────────────────────────────────────────────────────────────────

✅ DUPLICATE PREVENTION
   • If you drop the same file twice, it's only added once
   • No duplicate files in your list
   • Smart file path comparison

✅ 10-FILE LIMIT PROTECTION
   • If you have 8 files and try to add 3 more → Error
   • Error message shows: Current count + Attempted count
   • Files are NOT added if limit would be exceeded
   • Your existing list remains unchanged

✅ FOLDER BEHAVIOR
   • If you have files in list and add a folder → Files cleared
   • Folders and files cannot be mixed
   • This is by design (batch file limitation)

   def add_files(self, files):
       video_files = [f for f in files if self.is_video_file(f)]
       
       # Clear if current list is a folder
       if self.video_files and os.path.isdir(self.video_files[0]):
           self.video_files = []
       
       # Add new files, avoiding duplicates
       for video_file in video_files:
           if video_file not in self.video_files:
               self.video_files.append(video_file)  # APPENDS
       
       # Check 10-file limit
       if len(self.video_files) > 10:
           # Rollback and show error
